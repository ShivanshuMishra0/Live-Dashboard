# Live-Dashboard
A fully live, auto-updating weather dashboard built in a single HTML file. It runs in any browser, detects your location automatically, and pulls real-time data from OpenWeatherMap API
AtmosX — Live Weather Dashboard

A fully live, single-file weather dashboard powered by OpenWeatherMap API. No server, no installation, no frameworks — just open the HTML file in your browser and get real-time weather for your location instantly.


Preview
AtmosX is a premium dark-themed weather dashboard that auto-detects your location and displays real-time weather conditions, 7-day forecasts, air quality index, wind compass, UV index, and much more — all in a beautifully designed interface.

Features
Real-Time Weather Data

Current temperature with live °C reading
Today's Maximum and Minimum temperature
Feels Like temperature and Dew Point
Weather condition with dynamic icons
Cloud cover percentage
Humidity, Visibility, and Atmospheric Pressure
Precipitation amount in mm

7-Day Forecast

Full week forecast with weather icons
Max and Min temperature per day
Rain probability percentage per day
Today highlighted separately in the forecast strip

Temperature Trend Chart

Interactive dual-line chart (Max and Min over 7 days)
Orange line for maximum, Blue line for minimum
Hover tooltips showing exact values per day

Sun Position Tracker

Animated arc showing real-time sun position
Accurate Sunrise and Sunset times
Total day length in hours and minutes
Progress bar showing daylight remaining

UV Index

Live UV value fetched from OpenWeatherMap
Color-coded risk level bar
Labels: Low / Moderate / High / Very High / Extreme

Air Quality Index (AQI)

Animated doughnut gauge showing AQI level
Categories: Good / Fair / Moderate / Poor / Very Poor
Six individual pollutant readings:

PM10, PM2.5, O3 (Ozone), SO2, CO, NO2



Wind Compass

Live rotating compass needle pointing wind direction
Wind speed in kph
Gust speed, Sea Level Pressure, Ground Level Pressure
Cardinal direction badge (N, NE, SSW, etc.)

Rain Probability Chart

Horizontal bar chart for all 7 days
Percentage chance of rain per day

City Switcher

Quick switch between multiple cities
Pre-loaded live temperatures for each city tab
Defaults to your auto-detected location
